Most-used runtime setting keys (stored in `~/.kafclaw/timeline.db` settings):

- `daily_token_limit`
- `whatsapp_allowlist`
- `whatsapp_denylist`
- `whatsapp_pending`
- `whatsapp_pair_token`
- `silent_mode`
- `bot_repo_path`
- `selected_repo_path`
- `group_name`
- `group_active`
- `kafscale_lfs_proxy_url`

Primary config file:
- `~/.kafclaw/config.json`
